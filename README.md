# manjaroiso-t2: Manjaro Linux ISO for Macs with T2

You want the most unstable Arch-based distro for you shiny Intel Mac but your Mac has that T2 chip? Well here it is! Now you can install Manjaro on your Mac and experience unstable and self-breaking mess! Seriously, Manjaro is _[bad](https://manjarno.snorlax.sh)_. Use [EndeavourOS](https://github.com/t2linux/EndeavourOS-ISO-t2) for stable Arch-based easy-to-use distro. Still interested? Well, go to [releases](https://github.com/NoaHimesaka1873/manjaroiso-t2/releases) and grab newest ISO to see how bad it is.
